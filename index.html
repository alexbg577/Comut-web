Comut

L'application de partage privée pour vos groupes

Qu'est-ce que Comut?

Comut est une application mobile qui permet de partager des photos, vidéos et musiques en privé avec vos proches. Chaque groupe est totalement indépendant - ce qui est partagé dans un groupe reste dans ce groupe.

🔒 Privé

Chaque groupe est unique et isolé des autres. Vos données restent entre vous et vos proches.

📱 Hors-ligne

Fonctionne sans internet. Tout est synchronisé quand le réseau revient.

🎵 Shorts

Découvrez le contenu de votre groupe comme sur TikTok, avec musique aléatoire pour les photos.

📤 Gros fichiers

Partagez des fichiers jusqu'à 3GB. Upload en arrière-plan.

Fonctionnalités

4 Onglets principaux

📸 Publications - Voir toutes les photos, vidéos et musiques partagées. Trier par: tous, plus aimé, récent. Filtrer par type.

🎬 Shorts - Défilement vertical comme TikTok. Vidéos et photos avec musique.

❤️ Favoris - Retrouvez tous vos contenus favoris en un clin d'œil.

👤 Compte - Gérez votre profil, vos groupes et vos paramètres.

Système de groupes

En créant votre compte, vous pouvez:

  • Créer votre propre groupe (vous devenez owner)
  • Rejoindre un groupe existant avec un code
  • Gérer les membres (si vous êtes owner ou admin)

Interactions

❤️ Likes • 💬 Commentaires • ⭐ Favoris • 🔗 Partage (si activé par l'owner du groupe)

Comment utiliser Comut?

Télécharger l'app

Téléchargez l'APK Comut sur votre téléphone Android. Allez sur le repo GitHub et lancez le build EAS.

Voir sur GitHub

Créer un compte

Inscrivez-vous avec votre email, pseudo et mot de passe (8 caractères min). Vérifiez votre email.

Créer ou rejoindre un groupe

Créez votre groupe privé ou rejoignez-en un avec un code. Chaque groupe a son propre espace.

Partager du contenu

Uploadez photos, vidéos et musiques (max 3GB). L'upload se fait en arrière-plan.

Profiter des Shorts

Faites défiler les contenus comme sur TikTok. Les photos ont une musique aléatoire du groupe!

Pour les propriétaires de groupe

En tant qu'owner d'un groupe, vous pouvez:

  • Nommer des admins
  • Gérer les membres (ajouter/supprimer)
  • Activer/désactiver le partage externe
  • Supprimer le groupe

Note: Seul l'owner de Comut peut modifier les mots de passe des utilisateurs.
